# Rectangle

指定为经度和纬度坐标的二维区域。

```
new Cesium.Rectangle(west, south, east, north)
```

| 名称 | 类型 | 默认 | 描述 |
| :--- | :--- | :--- | :--- |
| `west` | 数 | `0.0` | 可选的最西经度，以弧度表示，范围为\[-Pi，Pi\]。 |
| `south` | 数 | `0.0` | 可选最南端的纬度，以弧度为单位，范围为\[-Pi / 2，Pi / 2\]。 |
| `east` | 数 | `0.0` | 可选的最东经度，以弧度表示，范围为\[-Pi，Pi\]。 |
| `north` | 数 | `0.0` | 可选最北纬度，以弧度表示，范围为\[-Pi / 2，Pi / 2\]。 |
