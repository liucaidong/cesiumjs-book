# Cartesian3

3D笛卡尔点。

```
new Cesium.Cartesian3(x, y, z)
```

| 名称 | 类型 | 默认 | 描述 |
| :--- | :--- | :--- | :--- |
| `x` | 数 | `0.0` | 可选的X组件。 |
| `y` | 数 | `0.0` | 可选Y组件。 |
| `z` | 数 | `0.0` | 可选Z组件。 |

#### 属性

#### distance {#.distance}

Cesium.Cartesian3.distance\(left, right\) → Number

计算两点之间的距离。
